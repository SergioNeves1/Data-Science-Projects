Seaborn is a python library for data visualization builds on the matplotlib library. It is used for statistical graphics.

In this python Seaborn tutorial part-11, We explained with a real-time example. How to draw seaborn heatmap correlation using the seaborn sns.heatmap() and df.corr() function with its parameters, keyword arguments and other functions in Hindi of “Who is responsible for global warming” & "Breast Cancer" DataFrame.

1. What is correlation?
2. How to get correlation matrix?
3. How to select best correlated features from large number of features (feature engineering/selection)?
4. How to use ax.tick_params() function?
