This video will talk through the concepts needed to understand a Hyperledger Fabric solution (first 23 minutes) and then go into the smart contract and client application code (last 17 minutes). The code features a sample Hyperledger Fabric web-app built with Vue.js, Node.js + Express, and the Hyperledger Fabric 1.4 Node.js SDK.
