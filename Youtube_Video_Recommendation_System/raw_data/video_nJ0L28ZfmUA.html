Published on Apr 9, 2019

In this video you will learn how to package your HTTP application code into a container and deploy it to Google Cloud Run, a serverless compute platform. Cloud Run is serverless, so it abstracts away the underlying infrastructure and runs and scales your stateless application automatically, even scaling down to zero when not in use.

For a tutorial version of this video, check out the quickstart: Build and deploy with Cloud Run.

Get started with Cloud Run at https://cloud.google.com/run
