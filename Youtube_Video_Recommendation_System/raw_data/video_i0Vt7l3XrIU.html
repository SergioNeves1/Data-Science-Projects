Published on Jan 11, 2020

A Kart Racing Game with Machine Learning Agents, or ML Agents in Unity - what does it look like? Let's take a look!

In this video we will learn how to implement Unity's machine learning toolkit ML Agents into Unity's own Kart Racing Game project. We'll use reinforcement learning to train our AI agent to travel around the track autonomously by seeing with raycasts and steering to avoid obstacles.

Download the free demo project here: https://ole.unity.com/mlkartproj

Learn more about ML Agents: https://ole.unity.com/mlagentsrepo

Learn more about Machine Learning at Unity: https://ole.unity.com/machinelearning
