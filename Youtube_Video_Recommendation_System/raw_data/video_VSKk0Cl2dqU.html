Published on Nov 23, 2018

Today’s video highlights a very simple 3D Model of a new home in Arizona. A DJI Mavic 2 Pro drone was used to orbit the home and take 49 images to be used in the 3D Model creation. The model was rendered with Pix4D Cloud and the desktop Pix4D Model software, Drone Deploy’s cloud, Maps Made Easy Cloud, and Photoscan Standard desktop solution. You’ll notice right away there are differences between each software package even thought the same exact data was used in all of the software packages.

Bottom line? 3D Modeling results will vary from software package to software package.
