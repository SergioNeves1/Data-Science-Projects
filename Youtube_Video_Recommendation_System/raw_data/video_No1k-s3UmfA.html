HASH.GUIDE brings you a video showing you how you can deploy a Crowdsale or ICO to POA's Core Network or POA Sokol Testnet. Connect to the network through MetaMask custom RPC and either deploy your own version of ICO Wizard so you can customize it or go to https://wizard.poa.network/ to deploy using their version. Deploying to POA network is just like running an ERC20 token on the Ethereum network except it is faster and cheaper to deploy. If you want to know more about POA Network, check out their website at https://poa.network

POA Core Network: https://core.poa.network
POA Sokol Testnet: https://sokol.poa.network
