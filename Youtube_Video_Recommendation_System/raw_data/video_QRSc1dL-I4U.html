Published on Sep 12, 2017

https://aws.amazon.com/lambda

Learning Objectives:
- Learn best practices for testing, debugging, and deploying serverless applications
- Understand how to use the AWS Serverless Application Model (AWS SAM) to model and deploy serverless applications
- Learn to use the AWS SAM Local CLI tool to locally test Lambda functions

When developing serverless applications, you need a way to locally test the AWS Lambda functions that make up the application before deploying them. Once you've finished debugging them locally, many developers choose to automate building and deploying their serverless application to the cloud. In this webinar, we share best practices for locally testing and then deploying your Lambda application using the AWS Serverless Application Model (AWS SAM) and AWS SAM Local. AWS SAM is an open source specification which defines simplified syntax for expressing functions, APIs, mappings, and DynamoDB tables for your application. You can use SAM templates to easily deploy your application with AWS CloudFormation. AWS SAM Local is a companion CLI tool which lets you locally test Lambda functions defined with AWS SAM.
