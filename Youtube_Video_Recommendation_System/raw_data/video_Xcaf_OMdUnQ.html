Published on Feb 15, 2020

Aprende como pasar de la ciencia ficción a la realidad en tan solo 8 pasos. En esta serie se describe como Crear, Entrenar, Testear y Publicar un modelo de Machine Learning en un servidor web local.

Para lograrlo se debe desarrollar el modelo de persistencia Machine Learning con joblib, luego crear una aplicación web con flask y por ultimo ejecutar por terminal el comando curl con el json de valores y el método POST.

Queda por hacer el deploy público a través de heroku para consumirlo desde la nube.
