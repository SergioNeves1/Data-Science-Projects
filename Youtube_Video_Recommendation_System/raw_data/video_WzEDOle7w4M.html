Machine Learning
===============
Data scientist:
==============
-- Responsible to train mathematical models.
-- Before training any model don’t have any knowledge. After training model it is able to take decisions on data
Data science is an approach, set of disciplines to train mathematical models
-- There is a model called model (math), before training a model doesn’t have any knowledge, after train it acquires knowledge
Once acquires knowledge, the knowledge needs to deploy in application. Then application becomes smart application.

-- To train the model, weapon is data
-- The extracted key patterns are called as knowledge
--There are 2 basic approaches to train model
1) Machine learning
2) Deep learning
-- Machine learning can recognize simple patterns
-- For complex patterns Deep learning can be used
