Published on Apr 15, 2019

Le Machine Learning est tout autour de vous. Les algorithmes d’apprentissage automatique et les réseaux de neurones font tourner YouTube, Google, Amazon, Facebook… la reconnaissance vocale, la voiture autonome, les hôpitaux, le système judiciaire. Le monde entier se tourne vers cette technologie. Mais pourquoi donc ?
Et bien il faut savoir qu’à l’origine, les ordinateurs ne sont pas intelligents. Ils sont juste doués pour faire des calculs. Mais tous les problèmes ne peuvent pas être résolu par des calculs : par exemple reconnaitre un chat depuis le signal électrique que notre œil nous envoie.
Mais le Machine Learning (apprentissage automatique) donne la capacité aux ordinateurs de dépasser cette limite en leur fournissant la capacité d’apprendre à faire une tâche à partir de données.
Pour cela, il existe ce qu’on appelle des algorithmes d’apprentissage, et il en existe 3 grandes familles :
- Le Supervised Learning (apprentissage supervisé) qui consiste à fournir à la machine des données étiquetées X, Y (labelled dataset) et la machine apprend alors le lien qui uni X et Y.
- Le Unsupervised Learning (apprentissage non supervisé) qui consiste à laisser la machine apprendre des structures dans les données X sans lui montrer Y
- Le Reinforcement Learning (Apprentissage par renforcement) qui consiste à récompenser la machine lorsqu’elle entreprend des bonnes actions pour qu’elle apprenne comment agir en autonomie.
Le Deep Learning (apprentissage profond) est une technique au sein même du machine Learning qui vise à développer des réseaux de neurones extrêmement larges pour en utilisant des milliards de données dans le but de dépasser les compétences humaines.
