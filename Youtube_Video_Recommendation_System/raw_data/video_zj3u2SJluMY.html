Published on Feb 15, 2020

See how SAP S/4HANA leverages machine learning applications to streamline the process of corporate purchasing. Starting with an employee ordering via image recognition, the order is approved by a manager, who leverages machine learning for accelerated decision making. By also enabling a proactive monitoring of the order execution, machine learning applications improve the purchasing process holistically and enhance the employee experience.
